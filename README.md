# Muszle
Muszle is a full-stack gym tracking platform designed to help users log workouts, track progress, and get AI-powered exercise recommendations — all synced to the cloud through Google Drive. The frontend is built with Next.js and React, styled with Tailwind CSS, and features Google OAuth authentication so each user gets a personal, secure experience from the moment they sign in. The interface is clean and dark-themed, with a date-based tabbed navigation system that lets users quickly flip between workout days and log exercises with optional weight inputs. It's designed to feel fast and natural, like a tool you'd actually want to open at the gym between sets.

The backend runs on Node.js with Express, handling all the heavy lifting behind the scenes. Workout data is stored as monthly-organized CSV files on the user's own Google Drive, making Google Drive the single source of truth for all fitness records. This cloud-native approach means data is never lost, syncs across devices automatically, and stays under the user's control. The backend also integrates with the Google Generative AI SDK (Gemini) to provide intelligent workout suggestions and exercise analysis, turning raw logged data into actionable fitness insights. User accounts and session management are handled through Google OAuth tokens, with a clean separation between authentication, data storage, and AI processing.

Muszle started as a personal project born from frustration with existing gym apps — most were either too bloated with features nobody uses, locked behind subscriptions, or stored data in proprietary formats with no export option. The idea was to build something minimal but complete: log your sets, see your history, get smart suggestions, and know that your data lives on your own Google Drive as simple CSV files you can open in any spreadsheet app. No vendor lock-in, no premium tiers, just a clean tool that does exactly what a gym tracker should do. The project represents one of the most technically ambitious builds in the portfolio, combining OAuth flows, cloud storage APIs, AI integration, and a responsive modern frontend into a cohesive product.
